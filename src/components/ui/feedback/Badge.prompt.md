Compact status marker. Tone maps to a Vials functional accent — never a large fill.

```jsx
<Badge tone="sos" dot>SOS · 24h rehab</Badge>
<Badge tone="warning">PAO 14d</Badge>
<Badge tone="safe">Safe for sun</Badge>
<Badge tone="info">Tracking</Badge>
```

Tones: `neutral` · `sos` (Cabernet) · `warning` (Amber) · `safe` (Green) · `info` (Cobalt) · `solid` (black). Mono type, uppercase by default.
